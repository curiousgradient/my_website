# Institution logos

Add your logos here for the "Worked & studied at" strip on the home page.

## BITS Pilani

Place your BITS Pilani logo as `bits-pilani.png` in this folder.

- Recommended format: PNG or SVG (transparent background)
- Recommended size: ~200px wide, any height (will be scaled to fit)

If the file is missing, the site will show "BITS Pilani" as text fallback.
